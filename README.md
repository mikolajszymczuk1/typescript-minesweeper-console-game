---
## TechStack:

- TypeScript
- Jest
- Gulp.js
- Node.js

---
## Setup project Guide:

### 1. Clone repo

### 2. Install packages
```
npm install
```
### 3. For development (run gulp watcher)
```
npm run dev
```
### 4. Build
```
npm run build
```
### 5. Run tests
```
npm run test
```
### 6. Start game
```
node dist/app.js
```
---
